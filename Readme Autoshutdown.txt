# ⏱️ Windows Shutdown Scheduler (Python)

This Python script allows you to **schedule a Windows shutdown** at a specific hour and minute that you input.

---

## 💻 Features

- 🔹 Simple command-line interface
- 🔹 Schedule shutdown at a specific time
- 🔹 Input validation and clean UX
- 🔹 Friendly output messages

---

## 🛠️ Requirements

- Python 3.x
- Windows OS (uses `shutdown` command)

---

## 🚀 How to Use

1. Open a terminal or command prompt
2. Run the script:
   ```bash
   python shutdown_scheduler.py
