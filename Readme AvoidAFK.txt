# 🖱️ Python Auto Mouse Mover

This script randomly moves the mouse cursor every few seconds to prevent the system from going idle or locking the screen.

---

## ⚙️ Features

- 📌 Moves cursor to a random position within a defined range
- 🕒 Works continuously with a delay (you can adjust)
- ✅ Helpful during long tasks or downloads when you don’t want the system to sleep

---

## 🛠️ Requirements

- Python 3.x
- [`pyautogui`](https://pypi.org/project/pyautogui/)

Install it using pip:

```bash
pip install pyautogui
